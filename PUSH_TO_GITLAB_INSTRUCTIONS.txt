STEP 1: Create GitLab Account & Repository
-------------------------------------------

1. Go to https://gitlab.com
2. Sign up for a free account (or log in if you have one)
3. Click the "+" button â†’ "New project/repository"
4. Choose "Create blank project"

7. COPY the repository URL (looks like: https://gitlab.com/YOUR_USERNAME/RE-UE4SS-Linux.git)

STEP 2: Run the Push Script
----------------------------

Open PowerShell in this directory and run:

    .\push-to-gitlab.ps1

It will:
- Ask for your GitLab repository URL
- Add it as remote
- Push all your code to GitLab
- Set up everything automatically

ALTERNATIVE: Manual Push
-------------------------

If the script doesn't work, you can push manually:

1. Add remote:
   git remote add origin https://gitlab.com/YOUR_USERNAME/YOUR_REPO.git

2. Set branch to main:
   git branch -M main

3. Push:
   git push -u origin main

================================================================================
OPTION B: GITHUB
================================================================================

STEP 1: Link GitHub to Epic Games (REQUIRED!)
-----------------------------------------------

IMPORTANT: Do this FIRST or the build will fail!

1. Go to https://www.epicgames.com/account/connections
2. Log in to your Epic Games account
3. Find GitHub and click "Connect"
4. Authorize the connection

This is required for Unreal Engine submodules to work.

STEP 2: Create GitHub Repository
---------------------------------

1. Go to https://github.com
2. Sign in (or create account)
3. Click "+" â†’ "New repository"
4. Fill in:
   - Repository name: RE-UE4SS-Linux (or whatever you want)
   - Visibility: Public or Private
   - DO NOT initialize with README (we already have files)
5. Click "Create repository"
6. COPY the repository URL (looks like: https://github.com/YOUR_USERNAME/RE-UE4SS-Linux.git)

STEP 3: Run the Push Script
----------------------------

Open PowerShell in this directory and run:

    .\push-to-github.ps1

It will:
- Ask for your GitHub repository URL

- Push all your code to GitHub
- Set up everything automatically
